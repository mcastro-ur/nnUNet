# Contributing to nnU-Net

## Which branch should I use?

Use the **`main`** branch for your local development work. It contains the latest stable code for this fork.

```bash
git clone https://github.com/mcastro-ur/nnUNet.git
cd nnUNet
git checkout main
```

Other branches (`option2`, `copilot/*`, etc.) are experimental or work-in-progress feature branches and should not be used as a base for new work unless you are explicitly working on that feature.

## Getting started

See the [installation instructions](documentation/installation_instructions.md) in the documentation folder for setup steps.
